## 차선 인식 및 제어

### 1. 목적 및 필요성

- 목적 : 차선을 정확히 인지하여 주행 중 방향지시등이 꺼져 있을 때, 차선 변경을 시도시, 차선 변경을 제한하여 다시 원래 차선으로 돌아오게 한다.
- 필요성 : 방향지시등을 켜지 않고 차선을 이동하여 발생하는 사고를 막기 위해서이다.
